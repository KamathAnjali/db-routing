This is a Database called climbing.
This Database consists of 2 tables.

These tables are:

1. mountain

2. climber


Table 1: Tablename is mountain
DDL query for mountain:
CREATE TABLE "mountain" (
"Mountain_ID" int,
"Name" text,
"Height" real,
"Prominence" real,
"Range" text,
"Country" text,
PRIMARY KEY ("Mountain_ID")
)


Table 2: Tablename is climber
DDL query for climber:
CREATE TABLE "climber" (
"Climber_ID" int,
"Name" text,
"Country" text,
"Time" text,
"Points" real,
"Mountain_ID" int,
PRIMARY KEY ("Climber_ID"),
FOREIGN KEY ("Mountain_ID") REFERENCES "mountain"("Mountain_ID")
)
