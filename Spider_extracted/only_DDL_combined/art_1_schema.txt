This is a Database called art_1.
This Database consists of 3 tables.

These tables are:

1. Artists (displayed as: artists)

2. Paintings (displayed as: paintings)

3. Sculptures (displayed as: sculptures)


Table 1: Tablename is Artists
Display name: artists
DDL query for Artists:
CREATE TABLE Artists (
  artistID INTEGER,
  lname TEXT,
  fname TEXT,
  birthYear INTEGER,
  deathYear INTEGER,
  PRIMARY KEY (artistID)
)


Table 2: Tablename is Paintings
Display name: paintings
DDL query for Paintings:
CREATE TABLE Paintings (
  paintingID INTEGER,
  title TEXT,
  year INTEGER,
  height_mm INTEGER,
  width_mm INTEGER,
  medium TEXT,
  mediumOn TEXT,
  location TEXT,
  painterID INTEGER,
  PRIMARY KEY (paintingID),
  FOREIGN KEY (painterID) REFERENCES Artists (artistID)
)


Table 3: Tablename is Sculptures
Display name: sculptures
DDL query for Sculptures:
CREATE TABLE Sculptures (
  sculptureID INTEGER,
  title TEXT,
  year INTEGER,
  medium TEXT,
  location TEXT,
  sculptorID INTEGER,
  PRIMARY KEY (sculptureID),
  FOREIGN KEY (sculptorID) REFERENCES Artists (artistID)
)
