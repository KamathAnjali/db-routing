This is a Database called mountain_photos.
This Database consists of 3 tables.

These tables are:

1. mountain

2. camera_lens (displayed as: camera lens)

3. photos


Table 1: Tablename is mountain
DDL query for mountain:
CREATE TABLE "mountain" (
"id" int,
"name" text,
"Height" real,
"Prominence" real,
"Range" text,
"Country" text,
primary key("id")
)


Table 2: Tablename is camera_lens
Display name: camera lens
DDL query for camera_lens:
CREATE TABLE "camera_lens" (
"id" int,
"brand" text,
"name" text,
"focal_length_mm" real,
"max_aperture" real,
primary key("id")
)


Table 3: Tablename is photos
DDL query for photos:
CREATE TABLE "photos" (
"id" int, 
"camera_lens_id" int,
"mountain_id" int,
"color" text, 
"name" text,
primary key("id"),
foreign key("camera_lens_id") references `camera_lens`("id"),
foreign key("mountain_id") references `mountain`("id")
)
