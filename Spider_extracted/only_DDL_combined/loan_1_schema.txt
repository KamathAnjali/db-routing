Table 2: Tablename is customer
DDL query for customer:
CREATE TABLE customer (
cust_ID varchar(3) PRIMARY KEY,
cust_name varchar(20),
acc_type char(1),
acc_bal int,
no_of_loans int,
credit_score int,

FOREIGN KEY(branch_ID) REFERENCES bank(branch_ID))
