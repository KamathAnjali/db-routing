This is a Database called restaurants.
This Database consists of 3 tables.

These tables are:

1. GEOGRAPHIC (displayed as: geographic)

2. RESTAURANT (displayed as: restaurant)

3. LOCATION (displayed as: location)


Table 1: Tablename is GEOGRAPHIC
Display name: geographic
DDL query for GEOGRAPHIC:
CREATE TABLE "GEOGRAPHIC" (
"CITY_NAME" text,
"COUNTY" text,
"REGION" text,
primary key("CITY_NAME")
)


Table 2: Tablename is RESTAURANT
Display name: restaurant
DDL query for RESTAURANT:
CREATE TABLE "RESTAURANT" (
"ID" int,
"NAME" text,
"FOOD_TYPE" text,
"CITY_NAME" text,
"RATING" real,
primary key("ID"),
foreign key ("CITY_NAME") references `GEOGRAPHIC`("CITY_NAME")
)


Table 3: Tablename is LOCATION
Display name: location
DDL query for LOCATION:
CREATE TABLE "LOCATION" (
"RESTAURANT_ID" int,
"HOUSE_NUMBER" int,
"STREET_NAME" text,
"CITY_NAME" text,
primary key("RESTAURANT_ID"),
foreign key ("CITY_NAME") references `GEOGRAPHIC`("CITY_NAME")
foreign key ("RESTAURANT_ID") references `RESTAURANT`("RESTAURANT_ID")
)
